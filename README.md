## Purpose

The purpose of this project is to provide a beginner-friendly, accessible resource for individuals seeking information on mental health. It aims to de-stigmatize mental health issues by presenting information on recognizing signs of stress and offering management techniques in a safe, non-judgmental digital environment.

## User Value

-   **For the External User:** The site offers an immediate, easy-to-navigate source of comfort and information. The layout is designed to be low-stress, with clear headings and an interactive FAQ section to help users find specific coping mechanisms quickly.
-   **For the Site Owner:** This project demonstrates the ability to build a responsive, empathetic user interface using modern front-end technologies while strictly adhering to accessibility standards.

## UX Design & Accessibility

-   **Color Scheme:** Used [e.g., Sage Green and White] to evoke a sense of calm and clarity.
-   **Navigation:** A sticky Bootstrap navbar ensures users can always find their way.
-   **Responsiveness:** Used Bootstrap 5 Grid system to ensure the site is fully functional on mobile, tablet, and desktop.
-   **Accessibility:** Semantic HTML5 tags were used. All images include `alt` text. High contrast ratios were maintained for readability.
